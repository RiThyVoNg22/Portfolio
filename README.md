# Mann Chanrithyvong - Portfolio Website

A professional portfolio website showcasing my skills, experience, education, and projects.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design matching the CV layout
- **Two-Column Layout**: 
  - Left sidebar with dark blue background containing profile, about me, contact, and skills
  - Right main content area with light gray background containing name, education, work experience, projects, and languages
- **Smooth Animations**: Fade-in animations on scroll for better user experience
- **Font Awesome Icons**: Professional icons for each section

## Technologies Used

- HTML5
- CSS3 (with Flexbox and responsive design)
- JavaScript (ES6+)
- Font Awesome 6.4.0 (for icons)

## Setup

1. Clone or download this repository
2. Open `index.html` in your web browser
3. For the profile picture, add an image named `profile.jpg` in the root directory, or the placeholder will be used

## File Structure

```
RithyVong.Portfolio/
├── index.html      # Main HTML file
├── styles.css      # CSS styling
├── script.js       # JavaScript for interactivity
├── profile.jpg     # Profile picture (add your own)
└── README.md       # This file
```

## Customization

- Replace `profile.jpg` with your own profile picture
- Update contact information, skills, or experience in `index.html`
- Modify colors and styling in `styles.css`
- Add additional interactive features in `script.js`
